## Installation
1. Clone the repository to your local machine:
   ```
   git clone https://github.com/izuchii/code-editor.git
   ```
2. Navigate to the project directory:
   ```
   cd code-editor
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Start the application:
   ```
   npm start
   ```
